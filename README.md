# Vote_Candidate

There are many websites which conduct compettition such as best baby photo, best photo, etc in which participate have to share the link among their members and ask them to vote in every certain time interval such as 30min, 1hour, etc. This application use to automate voting system just in one click.  
This application works on https://mycutebaby.in/contest/


### Prerequisites
1. Python 3.x.x
2. Pyinstaller
3. tkinter
4. selenium
5. Chrome webdriver


### Installing

To run this application you have to clone the repository and then make the binary file.
follow the step given below:


```
pyinstaller --onefile main.py
```
application place under dist folder with named main.exe

that its ready to use
